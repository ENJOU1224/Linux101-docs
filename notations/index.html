    记号约定¶

    对于在终端（命令行）环境下执行的连续几条命令，或者是需要展示输出结果的命令，用以下的方式表示：

    $ 某条命令
命令的输出

$ # 注释

# 需要使用 root 权限执行的命令

    这里：命令开头的 $ 代表此命令使用普通用户权限运行，命令开头的 # 代表此命令使用 root 权限（最高权限）运行，命令末尾的 # 代表此符号与其后的内容都是注释。
